# Проблема

Есть несколько офисов с кабинетами. Каждый кабинет владеет набором книг. Книги берут сотрудники и возращают через время.

Нужно иметь возможность учета книг по кабинетам и офисам. Также понимать какие книги сейчас заняты; какие пользователи изучали книги. Получать статистику по прочтению
